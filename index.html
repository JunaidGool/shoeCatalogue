Welcome to my Shoe Catalogue.

This is an online shoe catalogue which was designed for training purposes at www.projectcodex.co.za
Currently this catalogue allows for a stock controller to add stock, view quantities, view a bar chart showing total amount
of brands available and view lised items.
It also allows for a customer to access the "Shoe Catalogue" menu where all the available stock will be displayed.
The customer will be able to filter through the table to find the desired brands, size or color.

To begin, please select "Add Stock" which will allow you to add stock for the catalogue.
You can also select "Open Catalogue" to view the listed Shoes.

This catalogue specialzes in 4 brands, namely "Addidas", "Nike", "Puma" and Reebok.
